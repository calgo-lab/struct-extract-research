# struct-extract-research

Repository for research on machine learning–based methods for extracting structured information from unstructured documents. This repository can be considered a companion repository to [BHT_PDF](https://github.com/calgo-lab/BHT_PDF), which contains practical implementations based on research developed here.

## Getting Started (Locally)

*Version numbers are based on what has been tested/used. Other versions may work aswell.*

### Prerequisites
- Python 3.11.7 - Python runtime environment
- pip (version 25.3 or higher) - package manager

### Installation
#### Windows
`pip install -r requirements_windows.txt`
